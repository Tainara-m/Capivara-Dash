# 🐾 Capivara Dash

> Um jogo simples, divertido e dinâmico inspirado em mecânicas de corrida infinita.

---

## 📌 Sobre o Projeto

O **Capivara Dash** é um jogo desenvolvido para o navegador, onde o jogador controla uma capivara e precisa desviar de obstáculos enquanto acumula pontos.

⚠️ **Importante:**  
Este projeto foi desenvolvido **exclusivamente para fins de estudo**, com foco na prática de lógica de programação, manipulação do DOM e criação de interatividade com JavaScript.

---

## 🎯 Objetivos de Aprendizado

- Praticar lógica de jogos em JavaScript  
- Trabalhar com eventos e interações do usuário  
- Manipular elementos dinamicamente no DOM  
- Criar animações simples  
- Desenvolver uma experiência interativa no navegador  

---

## 🎮 Mecânica do Jogo

- 🐾 Controle de personagem (capivara)  
- ⛔ Obstáculos que surgem ao longo do percurso  
- ⏱️ Progressão contínua (estilo endless runner)  
- 🧮 Sistema de pontuação  
- ⚡ Aumento gradual de dificuldade (quando aplicável)  

---

## 🧭 Funcionalidades

- Início e reinício de jogo  
- Detecção de colisão  
- Contagem de pontos em tempo real  
- Feedback visual durante a partida  
- Interface simples e direta  

---

## 🎨 Interface

- Visual leve e divertido  
- Elementos gráficos simples  
- Foco na jogabilidade  
- Layout adaptado para diferentes telas  

---

## 📱 Responsividade

- Funciona em desktop  
- Ajustes básicos para dispositivos móveis  

---

## 🛠️ Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript  

---

## 💡 Principais Aprendizados

- Implementação de lógica de jogo  
- Controle de tempo e eventos

---

## ⚠️ Limitações

Por ser um projeto de estudo:

- Não possui sistema de níveis avançado  
- Não há armazenamento de pontuação  
- Mecânicas simplificadas  

---

## 🚀 Possíveis Melhorias

- Sistema de ranking (high score)  
- Sons e efeitos visuais  
- Diferentes níveis de dificuldade  
- Novos obstáculos  
- Melhor adaptação para mobile
